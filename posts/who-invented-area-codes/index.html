Who invented area codes?

The area code system was developed by AT and Bell Laboratories in the 1940’s, and went into effect in 1947. It was called the North American Numbering Plan (NANP) and included the United States and Canada .

When did we start using area codes?

On November 10, 1951, the official rollout of area codes took place.

Where does 487 area code come from?

Area Code 487 has not been assigned for use by the North American Numbering Plan Administrator. Area Code 487 is officially planned as a General Purpose area code which is typically assigned to a geographic area. If you receive a phone call from area code 487 it is a spam call.
